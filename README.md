# James Jiang Portfolio

Static GitHub Pages portfolio site for analytics and data product work.

## Local Preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8080
```

## Deploy

Push this repository to `jamesjiangyuhao/jamesjiangyuhao.github.io`. GitHub Pages will publish it at:

```text
https://jamesjiangyuhao.github.io/
```
